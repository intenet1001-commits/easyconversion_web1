PROJECT TYPE: Desktop Application (Electron + Next.js)
PLATFORM: macOS, Windows, Linux

UI LANGUAGE: Korean

================================================================================
                            CORE FEATURES
================================================================================

1. YOUTUBE DOWNLOAD
   ├─ MP4 (video + audio) download
   ├─ MP3 (audio only) download
   └─ Real-time progress tracking

2. MEDIA CONVERSION
   ├─ Formats: MP4, MP3, MOV, WAV
   ├─ Batch conversion support
   ├─ Multi-file processing
   └─ Progress tracking per file

3. MEDIA EDITING
   ├─ Split by timepoints or segments
   ├─ Merge multiple media files
   └─ Per-segment progress tracking

4. DOCUMENT CONVERSION
   ├─ PDF ↔ HTML/Markdown
   ├─ DOCX ↔ PDF/HTML/Markdown
   ├─ Markdown ↔ PDF/HTML/DOCX
   └─ ~12+ conversion paths supported

5. DOCUMENT OPERATIONS
   ├─ Merge PDFs
   ├─ Split by range, page count, heading
   └─ Batch processing

6. FILE MANAGEMENT
   ├─ Session-based uploads (24h retention)
   ├─ Drag-and-drop file reordering
   ├─ Auto-cleanup (24h+)
   └─ Storage monitoring

7. APP BUILDING
   ├─ DMG build for macOS
   ├─ Auto-install to Applications folder
   └─ Real-time build progress logs

================================================================================
                       DATA FLOW EXAMPLE
================================================================================

MEDIA CONVERSION FLOW:
  User Input
    ↓
  File Upload → /api/upload
    ↓ (saved to tmp/uploads/{sessionId}/)
  Zustand Store Update
    ↓
  User Clicks Convert
    ↓
  /api/media/convert Request
    ↓ Backend:
  ├─ Create output directory
  ├─ Spawn FFmpeg process
  ├─ Track progress
  └─ Stream progress via SSE (ReadableStream)
    ↓ Frontend:
  ├─ Receive SSE updates
  ├─ Update progress bar
  ├─ Log messages
  └─ Show download link
